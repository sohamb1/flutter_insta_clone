# Feedup: Instagram Clone

A Flutter Application having:
- Bottom Navigation
- Bloc architecture with BlocProvider, BlocConsumer, BlocListener
- Login with validation
- Http calls on REST APIs
- StaggeredListView
- Dependency Injection
- Router generator
- Custom AppBar
